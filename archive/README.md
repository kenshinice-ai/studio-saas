# StudioSaaS Archive

This folder keeps files that are useful for reference but are not part of the
active local StudioSaaS runtime.

## legacy-letspaint-reference

- `aws/` - old Lightsail and static IP deployment notes.
- `docs/` - old Let's Paint CMS manuals and usage notes.
- `data-snapshots/` - legacy JSON exports kept for migration reference.
- `runtime-legacy/` - old launch/runtime files no longer used by the active
  StudioSaaS structure.

## cleanup-clutter

Generated local clutter moved out of the active tree, including Python cache
folders and macOS `.DS_Store` files.
