# 每周汇总邮件 — 设置教程（5 分钟）

CMS 每周一早上 10:00 自动给你发一封经营汇总邮件（本周签到/充值/新学员、待续课名单、数据体检结果）。发件用你自己的 Gmail，需要一个「应用专用密码」——这是 Google 给第三方程序用的独立密码，随时可撤销，不影响你的主密码。

## 第一步：获取 Gmail 应用专用密码

1. 浏览器打开 **https://myaccount.google.com/apppasswords**（用 lee.liu.melbourne@gmail.com 登录）
2. 如果提示需要先开启「两步验证」：到 https://myaccount.google.com/security 开启两步验证后再回来（用手机号即可，一次性设置）
3. 在「应用专用密码」页面，App name 随便填（如 `LetsPaintCMS`），点 **创建**
4. Google 会显示一个 **16 位密码**（形如 `abcd efgh ijkl mnop`）——复制它（空格不用管，系统会自动处理）

> 安全说明：这个密码只存在你 Mac 上的 `.cms_config.json` 文件里，只能用来发邮件，不能登录你的 Google 账号。怀疑泄露时回到同一页面点删除即可立刻作废。

## 第二步：在 CMS 里配置

1. 打开管理后台 → 右上角 ⚙️ **系统设置** → 滚动到「📧 每周汇总邮件」
2. 填三项：
   - **收件邮箱**：lee.liu.melbourne@gmail.com（想发给谁都行，随时可改）
   - **发件 Gmail 地址**：lee.liu.melbourne@gmail.com
   - **应用专用密码**：粘贴第一步的 16 位密码
3. 打开「每周一自动发送」开关
4. 点 **保存配置**
5. 点 **发送测试邮件** → 去邮箱查收（测试邮件内含本周汇总的完整预览）

## 工作机制（了解即可）

- 每周一 **10:00** 后自动发送；如果那时 Mac 在睡眠/关机，醒来后会在本周内自动补发，不会漏
- 首次开启后，第一封邮件在**下周一**发出（不会立刻发）
- 发送失败会自动重试；连续失败 3 次暂停 6 小时再试（`./cms.sh logs` 可看到原因）
- 收不到邮件先查垃圾箱；Gmail 偶尔会把程序发的第一封邮件归到垃圾箱，标记"非垃圾邮件"后以后就正常了

## 常见问题

| 问题 | 解决 |
|---|---|
| 测试邮件报 `Username and Password not accepted` | 密码贴错或贴成了 Gmail 主密码——必须用 16 位应用专用密码 |
| 找不到「应用专用密码」入口 | 先开两步验证；公司/学校管理的账号可能被管理员禁用此功能 |
| 想换收件人 | 设置页改「收件邮箱」点保存即可，立即生效 |
| 想暂停 | 关掉「每周一自动发送」开关即可，配置保留 |
